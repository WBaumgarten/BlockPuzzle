# BlockPuzzle
Quick puzzle game about moving blocks with an easy way to create levels
Currently there is only one puzzle and no way to select others, puzzle selection is hardcoded.

Puzzle Template:
Rows.Cols
x.x.x.x.
x.x.x.x.
x.x.x.x.

Instead of "x" use the following:
f. = floor
w. = wall
p. = player
b. = box
g. = goal
